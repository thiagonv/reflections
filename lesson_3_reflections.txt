Quando gostaria de usar um repositório remoto em vez de manter o trabalho local?
- Para poder atualizar servidores, manter copias de segurança, compartilhar o
trabalho com outras pessoas.

Por que você prefere efetuar pull de alterações manualmente do que deixar o Git
atualizar-se automaticamente com o seu repositório remoto?
- Para ter o controle do que será enviado para o repositório remoto.

Descreva as diferenças entre bifurcações, clones e branches. Quando você usaria
um e não os outros?
- Bifurcações são clones dentro do github. Clones são realizados entre maquinas
diferentes. Branches são locais, são criadas durante o desenvolvimento.

Qual é o benefício de ter uma cópia do último estado conhecido do remoto
armazenado localmente?
- Ter uma copia atualizadas de todas as alteraçoes que está no repositorio remoto

Como você colaboraria sem usar Git ou GitHub? O que seria mais fácil e mais difícil?
- Enviando solitações por e-mail. É mais dificil controlar e testar as alteraçoes.
